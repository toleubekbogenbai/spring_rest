# Приложение Spring MVC / REST API с применением Hibernate
### Я использовал чистый Java код, базу данных mySQL.

--- 
* HTTP методы:
    * GET
    * POST
    * PUT
    * DELETE
___

### Поля в базе данных (mySQL)

    * id (int)
    * surname (String)
    * name (String)
    * department (String)
    * salary (String)
